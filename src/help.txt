Usage :           koala                   Run Koala Toolkit
                  koala [argument]        Options

Arguments :
  -b :            Build Koala Toolkit
  -e :            Edit Koala Dockerfile
  -h :            Print help
  -k :            Kill all Koala containers
  -r :            Remove Koala image
  -uninstall :    Uninstall Koala
  -v :            Print Version
